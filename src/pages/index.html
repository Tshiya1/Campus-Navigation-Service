Navigation Service Documentation V1

GET /poi

Description: Retrieves a list of points of interest.

Parameters: None

Response:

                    [
                    {
                    "name" : string,
                    "id" : string,
                    "coordinates" : { "latitude" : number,
                    "longitude":number}
                    
                    }
                    ]

GET /poi/{id}

Description: Retrieves a point of interest by it's Id.

Parameters: 

{
    "id": string
}

Response:

{
                    "name" : string,
                    "id" : string,
                    "coordinates" : { "latitude" : number,
                    "longtitude":number}
                    
                    }
